Привет, мир!

Этот текст отображается с помощью JavaScript.

Как это работает:

!DOCTYPE html: Объявление типа документа HTML5.

html: Корневой элемент HTML-страницы.

head: Содержит метаданные о странице (например, заголовок).

title: Заголовок вкладки браузера.

style: Встроенные стили CSS для центрирования текста и задания шрифта.

body: Содержит видимое содержимое страницы.

h1: Заголовок первого уровня.

p id="output": Параграф с id="output". Этот id используется JavaScript для доступа к элементу.

script: Включает JavaScript код.

document.getElementById("output"): Получает HTML-элемент с id="output".

.textContent = "Это сообщение сгенерировано JavaScript!";: Изменяет текст внутри элемента, найденного выше.

Как запустить:

Сохраните этот код в файл с расширением .html (например, привет.html).

Откройте файл привет.html в любом современном веб-браузере (Chrome, Firefox, Safari, Edge и т.д.).

Браузер интерпретирует HTML и выполнит JavaScript код, отобразив "Привет, мир!" и текст, сгенерированный JavaScript.

Почему это лучше, чем Java Applets:

Совместимость: Работает во всех современных браузерах без необходимости установки дополнительных плагинов.

Безопасность: JavaScript имеет гораздо более строгие ограничения безопасности, чем Java Applets.

Производительность: JavaScript часто более производителен, особенно для простых задач.

Стандарты: HTML, CSS и JavaScript являются стандартами веб-разработки.

Простота разработки: Разработка на HTML и JavaScript обычно проще и быстрее, чем разработка Java Applets.

Этот простой пример показывает, как можно добиться интерактивности на веб-странице без использования устаревших технологий, таких как Java Applets.
